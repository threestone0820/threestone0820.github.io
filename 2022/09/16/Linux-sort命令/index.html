Linux sort命令

Posted on 2022-09-16 In bash

一些常用的参数

 -t char, --field-separator=char
             Use char as a field separator character.
 -k field1[,field2], --key=field1[,field2]
             Define a restricted sort key that has the starting position field1, and
             optional ending position field2 of a key field.
 -n, --numeric-sort, --sort=numeric
             Sort fields numerically by arithmetic value.
 -r, --reverse
             Sort in reverse order.

例子：按照某个field排序

ls 5_* | sort -t '_' -k2 -n
